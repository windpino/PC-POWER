원격 부팅 & 종료 설정 가이드

컴퓨터를 외부에서 안정적으로 켜고 끄기 위해 아래 단계들을 차례로 설정해 주세요.

01

BIOS 메인보드 설정 (컴퓨터 켜기 기본)

PC 전원이 완전히 꺼진 상태에서 전원을 켜려면 메인보드 BIOS 설정이 필요합니다.

  • 부팅 시 Del 또는 F2 키를 입력하여 BIOS 모드로 진입합니다.
  • 방법 A: Wake-on-LAN 설정 (추천)
    • Advanced / Power Management Setup 메뉴로 이동합니다.
    • Power On By PCI-E 또는 Wake on LAN 설정을 Enabled로 변경합니다.
    • ErP Ready 설정이 있다면 Disabled로 설정해 전원 공급을 유지합니다.
  • 방법 B: 스마트 플러그 연동 (자동 켜기)
    • Power Management 메뉴에서 Restore on AC Power Loss (또는 AC Back) 설정을 Always On 또는 Last State로 변경합니다.
    • 이 설정을 켜면 스마트 플러그 전원을 차단했다가 다시 켤 때 PC가 자동으로 부팅됩니다.
02

윈도우 랜카드 설정 (Wake-on-LAN 필수)

윈도우가 종료된 상태에서도 랜카드가 대기 전력을 받으며 신호를 기다리도록 설정합니다.

  1. 윈도우 시작 버튼 우클릭 -> 장치 관리자를 실행합니다.
  2. 네트워크 어댑터 항목을 펼치고 사용 중인 유선 랜카드(예: Realtek, Intel)를 더블 클릭합니다.
  3. [전원 관리] 탭에서 이 장치를 사용하여 컴퓨터를 대기 상태에서 깨움 및 매직 패킷에서만 컴퓨터를 깨움에 체크합니다.
  4. [고급] 탭으로 이동하여 아래 속성들을 활성화(Enabled)합니다.
    • 웨이크 온 랜 (Wake on Magic Packet) -> 활성화(Enabled)
    • 절전형 이더넷 (Energy Efficient Ethernet) -> 비활성화(Disabled) (원치 않는 해제 방지)
03

에이전트 프로그램 실행 (종료 및 페어링용)

제어하고 싶은 컴퓨터에 에이전트 실행 파일(`PC_Power_Setup.exe`)을 배치하고 더블 클릭하여 실행합니다.

  1. 프로그램이 처음 실행되면 검은 콘솔창에 6자리 인증번호(PIN)가 표시됩니다.
  2. 웹 대시보드에서 [PC 추가] 버튼을 누르고 이 6자리 숫자를 입력합니다.
  3. 연동이 성공하면 에이전트가 조용히 화면에서 사라져 백그라운드 서비스(시작프로그램)로 자동 등록됩니다.
  4. 연동 직후 대시보드 화면에 해당 PC가 **ONLINE** 상태로 나타납니다.
04

원격으로 PC 켜기 (WOL 브릿지 설정)

PC가 완전히 꺼지면 스스로 인터넷을 통해 부팅 신호를 받을 수 없습니다. 아래 중 하나의 방법으로 전원을 켭니다.

방법 A: WOL 브릿지 에이전트 (권장)

공유기에 연결된 항상 켜진 기기(라즈베리 파이, 항상 켜진 구형 폰, 홈서버 등)에 bridge.js를 실행해 둡니다. 웹 대시보드에서 켜기 버튼을 누르면 이 브릿지 기기가 대신 WOL 패킷을 날려 PC를 깨웁니다.

방법 B: 스마트 플러그 활용

BIOS에서 AC Back을 켜고, 스마트 플러그 전원을 차단(OFF)했다가 다시 켬(ON)으로써 PC를 구동합니다. Tuya, Tapo 등의 자동화 규칙으로 편리하게 제어 가능합니다.

방법 C: 공유기 WOL 직접 연결

iptime 등 공유기의 관리 페이지에 접속해 외부 WOL 기능을 활성화하고 DDNS 주소를 확보한 경우, 외부에서 공유기에 접속하여 WOL 패킷을 보냅니다.
